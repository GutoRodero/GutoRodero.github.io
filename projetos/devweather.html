O problema

Aplicativos de clima são cheios de anúncios, popups e informações irrelevantes. Devs preferem interfaces limpas — nada mais limpo que um terminal.

A solução

Uma dashboard/CLI web que consome a OpenWeather API e apresenta os dados em formato ASCII art. Suporta comandos estilo Unix (weather, forecast, help, cities). Salva favoritos localmente e usa gráficos em SVG para tendências.

Features principais

⌨️

CLI real

Comandos como weather, forecast, help — parseados como um shell.

📈

Gráficos ASCII

Tendências de temperatura e chuva em ASCII art, direto no terminal.

⭐

Favoritos

Salve suas cidades preferidas para acesso rápido.

🌐

Multi-linguagem

Português e inglês. Comandos aceitam ambas as línguas.

📴

Cache offline

Última consulta fica salva mesmo sem internet.

🎨

Temas ANSI

Solarized, Dracula, Nord, Monokai — pick your poison.

O que aprendi

Escrever um parser de comandos foi um exercício divertido de expressões regulares e árvore de estados. Aprendi também a lidar com rate limits de APIs públicas, cache com service workers e como fazer arte ASCII programaticamente.
